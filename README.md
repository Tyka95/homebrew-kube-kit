# Homebrew Tap for KubeKit

## Install

```bash
brew tap Tyka95/kube-kit
brew install kubekit
```

## Upgrade

```bash
brew upgrade kubekit
```

## Uninstall

```bash
brew uninstall kubekit
brew untap Tyka95/kube-kit
```

## About

This tap distributes [KubeKit](https://github.com/Tyka95/kube-kit) — an interactive terminal UI toolkit for Kubernetes and AWS workflows.
